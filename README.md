# Clinical Breast Cancer Survival Analysis
This repository builds an end-to-end survival analysis pipeline using breat cancer clinical data.

## Goal
To understand each step of a survival analysis workflow in R, including:
- Data Cleaning
- Kaplan-Meier estimation
- Cox Modelling
- Diagnostics
- Validation
- Machine Learning extensions

## Dataset
Clinical breast cancer survival data with:
- overall survival
- relapse-free survival
- treatment variables
- prognostic markers
